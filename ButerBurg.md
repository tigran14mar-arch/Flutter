# ButerBurg — мобильное приложение для кафе

Приложение для кафе, которое позволяет клиентам заказывать еду онлайн, а администраторам — управлять заказами в реальном времени.

## Обзор

**ButerBurg** упрощает процесс заказа еды в кафе: клиенты выбирают блюда из меню, оформляют заказы, а администраторы видят их в панели управления и могут менять статусы (новый, готовится, готов, выполнен).

### Для клиентов:
* просмотр меню с категориями и фильтрами;
* добавление блюд в корзину;
* оформление заказа с выбором способа получения (на месте / самовывоз);
* отслеживание статуса заказа в реальном времени;
* история заказов.

### Для администраторов:
* панель управления заказами;
* обновление статусов заказов;
* просмотр деталей заказа (состав, время, способ получения);
* уведомления о новых заказах.

## Технологии и стек

**Клиентская часть (мобильное приложение):**
* **Flutter** — кроссплатформенная разработка (iOS и Android);
* **Dart** — язык программирования;
* **HTTP / WEBSOCKET** — взаимодействие с бэкендом;
* **SharedPreferences** — локальное хранение данных (токены, настройки);
* **Firebase Cloud Messaging (FCM)** — push‑уведомления.

**Серверная часть:**
* **Python** — бэкенд‑сервер;
* **MySQL** — база данных;
